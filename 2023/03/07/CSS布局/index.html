CSS布局

发表于2023-03-07|更新于2023-03-20|前端CSS
|阅读量:

元素定位

​ 标准流（Normal Flow）：浏览器默认的布局形式，从左到右从上到下，不存在层叠，可以通过margin，padding控制元素位置，但是因为不层叠也会影响其他元素的位置

position脱离标准流

常见5个取值设置后四个值后元素变为定位元素，left，right，top，bottom才有效果

  1. static：默认值，静态定位
  2. relative：相对定位（相对于自己原来的位置）不脱标
  3. fixed：固定定位（相对于浏览器视口，不占空间位置）
  4. absolute：绝对定位（相对于最邻近定位祖先元素）
  5. sticky：粘性定位

粘性定位：设定到最近的祖先滚动容器固定位置变成绝对定位(兼容性略差)

absolute和fixed元素特点：

  1. 可以随意设置宽高
  2. 不向父元素汇报宽高
  3. 不再区分块级行内级元素
  4. 脱标元素内部仍然按标准流布局

定位元素属性

  1. z-index：定位元素重叠时显示的优先级(默认按出现顺序依次层叠)
    优先级只比较兄弟元素

CSS浮动

沿容器靠左或靠右显示，允许被内联元素环绕

  • float：none（不浮动），left（向左浮动），right（向右浮动）

浮动规则

  1. 浮动元素脱标，移动到贴到父元素上（或其他浮动元素边界），定位元素会层叠在浮动元素上
  2. 浮动不能超出包含块边界
  3. 浮动元素之间不能层叠
  4. 行内级元素内容会被浮动元素推出来（有没有脱标的效果，图文环绕效果）
  5. inline，inline-block元素浮动顶部会和行顶部对齐

！高度塌陷问题：由于浮动元素脱标不向父元素汇报宽高，导致后面元素有可能会上移到不是自己想设置的位置

解决办法：CSS属性clear：left（移动到上面所有向左浮动元素下面），right（移动到上面所有向右浮动元素下面），both（上面两个属性同时满足），可以用元素实现，可以用伪元素实现

flex布局

设置为flex的盒子为container，直接子元素为item(弹性盒子)

flex布局的结构：main axis主轴，cross axis交叉轴

flex属性：

container属性

  • flex-direction：决定主轴方向，row（行）,row-reverse(行反转)，column（列），column-reverse（列反转）
  • flex-wrap：设置换行情况，wrap（多行），wrap-reverse（多行从下往上排）
  • flex-flow：前两个任意顺序设置值的简写属性
  • justify-content：对item之间对齐排布方式的设置
    1. flex-start：与main-start对齐
    2. flex-end：与main-end对齐
    3. center：居中对齐
    4. space-between：item之间间距相等，与main-start和main-end对齐
    5. space-evenly：space-between基础上加上两边也要间距
    6. space-around：space-evenly基础上两端间距为一半
  • align-item：对item在交叉轴上的对齐方式设置
    1. flex-start：与cross-start对齐
    2. flex-end：与cross-end对齐
    3. center：居中对齐
    4. baseline：item的基线对齐（默认和flex-start效果一样，但是每个item的基线本来就不在一条线上时效果就不一样，比如item内文字大小不一样）
    5. normal：等价于stretch，height默认时为拉伸效果
  • align-content：多行，行与行之间在交叉轴上的对齐方式
    1. 设置属性与justify-content一致，效果类比一下
    2. stretch：height默认时纵向拉伸

item属性

  • order：item的属性，决定了item排布顺序，设置的值越小越靠前

  • align-self：给某个item单独设置（align-item一样）的属性

  • flex-grow：设置拉伸属性（可设置0，正小数，正整数），默认0不拉伸，只有主轴方向上有空余size是才有效，设置的值为拉伸权重，比如三个元素设置为1，1，2，那么空余的空间就会加权分给这三个item

  • flex-shrink：设置收缩属性(默认1拉伸)，与grow一样设置值为压缩的加权值，item总size超出container才有效，设置为0则超出container

  • min-height，min-width：压缩后不能小于这两个值

  • flex-basis：基础宽度，应用场景：当英文单词超出界时会自动增加长度显示全

  • flex：flex-grow，flex-shrink，flex-basis简写属性

JavaScript语法

发表于2023-03-30|更新于2023-06-25
|阅读量:

JS交互方式

  1. alert函数：弹窗形式
  2. console.log函数：将内容输出到控制台
  3. document.write函数：将内容写入文档
  4. promp函数：弹窗获取用户输入并以字符串返回

JS的文档注释

js的文档注释可以在定义函数之前使用，帮助解释函数作用，参数类型等信息

JS基础语法（不同于其他编程语言之处）

  • 定义变量的关键字全部都是var
  • 变量由字母，_，**$**，开头
  • 变量类型是动态类型，即前一刻赋值字符串，后一刻可以赋值整数number类型有两个特殊值：infinity(无穷大)，NaN（计算错误：比如字符串和数字相乘）

    数据类型

  1. Number

    number类型有两个特殊值：infinity(无穷大)，NaN（计算错误：比如字符串和数字相乘）

    isNaN()：判断是否为数字，返回值bool类型

  2. String

    单引号双引号没区别，反引号可以执行里面的${}内容实现字符串拼接，很类似shell编程的用法，可能是脚本语言的共同特征吧hh

  3. undefined

    没有定义类型：值也是undefined，不常用

数据类型转换

  1. 两个变量相加，有一边是字符串，另一边自动转为字符串
  2. 两个字符串做 ”+“ ”-“ ”*“运算是会将两边字符串转为Number类型再运算
  1. 直观上为空的值转bool类型都是false，其余都是true。注意：”0“字符串转为true

判等运算符 “==” 和 “===”

  1. ==运算符在类型不同时会将两边全部转换为Number类型进行比较，NULL特殊除外
  2. ===运算符如果两边类型不同直接返回false

不等的用法类似

函数

JS中函数是头等公民（函数式编程）

  1. 函数可以被赋值给变量（函数表达式 ）
  2. 函数可以在变量之间传递
  3. 函数可以作为另一个函数的参数
  4. 函数作为另外一个函数的返回值
  5. 函数存放在另一个数据结构中

总之，上面五种特点像是把函数当成变量来对待

立即执行函数：与普通代码块区别是有自己的作用域，防止全局变量命名冲突，

面向对象

其他语言的构造函数是类中的一个方法，JS中构造函数在其他语言中扮演类的角色

在ES6之后可以像别的语言用class声明类

开发规范

  1. 变量命名（驼峰命名法）

    一个变量由很多字母组成，从第二个字母开始首字母大写

    但是构造函数为了和普通函数区分采用大驼峰命名

  2. 赋值操作两边都加空格

  3. 加不加分号：看公司习惯

  4. 程序员约定俗成一些常用名词，伪变量：foo，bar，baz

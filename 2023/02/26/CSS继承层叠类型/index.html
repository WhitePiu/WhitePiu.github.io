CSS继承层叠类型

发表于2023-02-26|更新于2023-03-01|前端CSS
|阅读量:

继承

对于可继承的属性，父元素的设置属性子元素会共享

层叠

选择器属性优先级：

  • important 10000
  • 内敛样式 1000
  • id选择器 100
  • 类选择器，伪类选择器，属性选择器 10
  • 元素选择器，伪元素选择器 1
  • 通配符 0

HTML元素类型

  1. 块级元素 block：独占一行，可以设置长宽
  2. 行内级元素 inline ：不独占，不能设置长宽
  3. 行内块级 inline-block ：不独占，可以设置长款
  4. none ：不显示，不占地方，没打小（相当于没有）

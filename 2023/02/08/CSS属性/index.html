CSS属性

发表于2023-02-08|更新于2023-03-30|前端CSS
|阅读量:

css官方文档

css推荐文档

常见CSS属性

  • font-size 设置文字大小
  • color 前景色（文字颜色）
  • background-color 背景色
  • width 宽度
  • height 高度

CSS文本属性

  • text-decoration 文本加线段

  • text-transform 英文文本大小写控制（一般不用，用JavaScript控制）

  • text-indent 段落缩进

  • text-align 行内元素对齐方式（文本，图片等） 原理很重要！

  • letter-spacing 字母间距

  • word-spacing 单词间距

CSS字体属性

  • font-size 字体大小设置，可用px，em，%（后两者相对继承下来父元素的大小）

  • font-family 字体设置，一般只设置一次

  • font-weight 字体粗细设置，一般为100~900

  • font-style 字体样子设置（倾斜）

  • font-variant 改变小写字母显示形式

  • line-height 两行文字基线的距离（有让文本居中的效果）

font属性可以缩写，写法为 font： font-style font-variant font-weight font-size/line-height font-family

CSS背景属性

  • background-image 背景图片
  • background-repeat 背景图片平铺情况
  • background-size 背景大小尺寸
    1. auto：默认大小
    2. cover：覆盖整个背景，但保持图片长宽比，所有有可能会有部分看不见
    3. contain：只满足达到长或宽中的一个，保持长宽比，图片全能显示
    4. 百分比：100% 100%也会覆盖整个背景，所以背景长宽比和图片长宽比不同时，图片肯定会变形
    5. ..px ..px具体长宽
  • background-position 背景位置
  • background-attachment
    1. scroll：默认值，不随文字滚动
    2. local：当文字滚动时，图片会随着文字滚动滚动
    3. fixed：浏览器滚动也不滚，跟随浏览器视口
  • background 缩写

vertical-align

一个行盒内行内级元素垂直方向对齐方式

行盒：对于一个盒子内部每行可以理解为都有一个隐藏的行盒，此行盒完全包含这一行内所有元素

默认基线对齐（baseline）

文本基线位置：x最底部

img基线位置：最底部

inline-block盒子：没有文本时是底部，有文本时是最后一行文本的基线

vertical-align是盒子内元素设置的对齐方式，不是父元素设置

  • baseline：默认
  • top：顶部对齐
  • middle：中线对齐
  • bottom：底部对齐

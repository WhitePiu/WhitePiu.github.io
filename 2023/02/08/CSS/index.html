CSS编写规则

内联样式

内联样式表存在于HTML元素的style属性中（全局属性）

内部样式

为解决要添加太多样式导致阅读性差的问题，增添了内部样式，在head里添加style

主要为了给元素添加样式，用选择器选择给具体哪个或者哪些元素添加

外部样式

解决多个html文件样式有相近部分时有全局的编号的样式文件拱其调用使用，类似编程语言的头文件的作用

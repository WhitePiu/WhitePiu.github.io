CSS盒子模型

发表于2023-02-28|更新于2023-03-01|前端CSS
|阅读量:

内容

设置width，height，min-width，max-width

min-width是最小宽度，缩小浏览器到一定程度会以滚动条实现

max-width是最大宽度，扩大浏览器到一定程度元素不在覆盖多出部分

auto：特殊设置值，自动随元素内容和浏览器大小而变

width，height默认值auto

内边距

设置可以通过padding

height-line行高可以设置上下边距（本质上是改变元素内容，并没设置内边距）

padding缩写属性：三个值省略左，和右一样，两个值省略下，和上一样，一个值即为所有内边距

边框

设置width，style，color

语法与padding一致

圆角设置：

  1. px设置
  2. 百分比设置：横向纵向有可能不一样，相对于边框宽度与内容宽度的和
  3. 当正方形时，百分比超50%就是圆

外边距

margin：设置上下左右外边距，与padding语法一致，目前左右元素变距会叠加，上下元素变距会重叠

外边距存在传递现象：

  • margin-top传递：块级元素顶部线与父级元素重合，子元素的margin-top传递给父元素
  • margin-bottom传递：块级元素底部线与父级元素重合，and父元素高度为auto

解决办法：父元素设置padding顶部/底部，父元素设置border，触发BFS（设置overflow）

外轮廓

outline，有样式，大小，颜色，基本和border一致，但是不占用空间，一般用作把选中的伪类外轮廓删除

盒子阴影

box-shadow：为内容设置阴影

在线调试阴影网站
